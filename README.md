🌤️ Skyline Weather Dashboard

A modern, responsive Weather Dashboard built with **HTML5, CSS3, and Vanilla JavaScript** that provides real-time weather information using the **OpenWeatherMap REST API**. Users can search for any city, view current weather conditions, hourly and 5-day forecasts, and additional weather metrics through a clean and interactive interface.

---

## 🚀 Live Demo

🔗 **Live Website:** https://bright-syrniki-2e4aae.netlify.app

🔗 **GitHub Repository:** https://github.com/Gyani1205/WEATHER-APP

---

## 📌 Features

- 🌍 Search weather by city name
- 📍 Current location weather using Geolocation API
- 🌡️ Real-time temperature
- 💧 Humidity
- 🌬️ Wind Speed
- 📊 Atmospheric Pressure
- 👁️ Visibility
- 🌤️ Hourly Forecast
- 📅 5-Day Forecast
- ☀️ UV Index
- 🌫️ Air Quality Index (AQI)
- 🕒 Search History using Local Storage
- 🌙 Dark / Light Theme
- ⚠️ Error Handling for invalid city names and network failures
- 📱 Fully Responsive Design

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Gyani1205/WEATHER-APP.git
```

### 2. Open the project

```bash
cd WEATHER-APP
```

### 3. Configure API Key

Create a file named `config.js` and add your OpenWeatherMap API key:

```javascript
export const API_KEY = "YOUR_OPENWEATHER_API_KEY";
```

Alternatively, replace the placeholder API key if your project uses one directly.

### 4. Run the project

Open `index.html` using **Live Server** in VS Code.
