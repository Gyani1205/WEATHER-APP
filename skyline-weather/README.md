# Skyline Weather Dashboard

## Overview

Skyline Weather Dashboard is a responsive web application that provides
real-time weather information using the OpenWeatherMap REST API.

Users can search weather by city name, use their current location,
view hourly and 5-day forecasts, air quality, UV index and more.

---

## Features

- Current Weather
- Hourly Forecast
- 5-Day Forecast
- Air Quality
- UV Index
- Humidity
- Wind Speed
- Pressure
- Visibility
- Geolocation
- Search History
- Dark/Light Theme
- Responsive Design
- Error Handling

---

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- Async/Await
- OpenWeatherMap REST API

## Installation

```bash
git clone YOUR_REPO
cd skyline-weather
```

Open

```
index.html
```

using Live Server.

---

## API Setup

Create an OpenWeatherMap API key.

Replace

```javascript
const API_KEY="YOUR_OPENWEATHER_API_KEY";
```

with your own key.
